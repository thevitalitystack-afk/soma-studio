# CLIPT Engine - YouTube Shorts Pipeline

## 🎬 Clip Engine Overview

**Purpose:** Automated YouTube Shorts production pipeline
**Target:** 30-60 second viral content
**Niche:** Spirituality, wellness, mindfulness
**Goal:** High retention, top-of-funnel growth

## ⚙️ Pipeline Architecture

### Phase 1: Trend Analysis
**Agent:** BEACON
- Scan trending spirituality topics
- Analyze competitor performance
- Identify viral hooks
- Generate title variants

### Phase 2: Concept Development
**Agent:** HELM
- Select top 3 concepts
- Define hook strategy
- Approve content direction
- Lock creative brief

### Phase 3: Script Creation
**Agent:** VERSE
- Write hook (0-3 seconds)
- Create 9-second beats
- Develop close + CTA
- Format for voiceover

### Phase 4: Voice Generation
**Agent:** CHORUS
- Generate ElevenLabs audio
- Apply voice parameters
- Sync with script timing
- Export clean audio

### Phase 5: Video Editing
**Agent:** BLADE
- Cut pacing (tight, no dead air)
- Add b-roll cues
- Insert text overlays
- Apply teal League Spartan captions

### Phase 6: Thumbnail Design
**Agent:** FRAME
- Create 1600x1600 cover
- Apply dark + ember aesthetic
- Add teal accent text
- Test A/B variants

## 🎯 Shorts Structure

### Hook (0-3 seconds)
- Punchy beats
- Teal caption overlay
- Immediate attention grab

### Content (3-45 seconds)
- 3-4 quick beats
- Visual storytelling
- Minimal text
- High energy

### Close + CTA (45-60 seconds)
- Direct call to action
- Engagement prompt
- Subscribe reminder
- Emoji: 🌿
