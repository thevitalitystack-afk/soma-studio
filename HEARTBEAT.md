# SOMA Studio Heartbeats

## Daily Checks (Every 4 hours during waking hours)

### YouTube Pipeline Health
- Check agent statuses (BEACON, HELM, VERSE, CHORUS, BLADE, FRAME)
- Verify research output quality
- Monitor video performance metrics
- Check dashboard for errors

### Content Queue
- Review pending scripts
- Check thumbnail concepts
- Verify upload queue status
- Monitor publishing schedule

### Revenue Tracking
- Check Gumroad sales
- Review Etsy orders
- Monitor affiliate commissions
- Update revenue dashboard

## Hourly Checks (During active production)

### Agent Activity
- BEACON: Research completion rate
- VERSE: Script writing progress
- BLADE: Edit completion status
- FRAME: Thumbnail generation

### System Status
- Dashboard connectivity
- Voice synthesis quality
- File storage availability
- API rate limits

## Trigger Conditions

### Immediate Alerts
- Agent offline for >30 min
- Revenue drop >20%
- Error rate >5%
- Dashboard unresponsive

### Morning Brief
- 9 AM ADT: Daily pipeline summary
- 12 PM ADT: Mid-day progress check
- 3 PM ADT: Performance review
- 6 PM ADT: End-of-day wrap

## Memory Updates
- Update pipeline performance in MEMORY.md
- Log successful agent outputs
- Document system improvements
- Track revenue milestones
